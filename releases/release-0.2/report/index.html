Suite MemoryManager create

MemoryManager creating new buffers. This shows how expensive allocation direct mapped buffers is. The default buffer size is 10MB and therefore buffer allocation much rarer.

create Buffer (100 bytes)Allocates blocks so that every block needs a new buffer. Buffer size is 100 bytes.
create Buffer (1000 bytes)Allocates blocks so that every block needs a new buffer. Buffer size is 1000 bytes.

Input ranges from 0 to 1000 in 100 steps.

Additional information and statistics about measurements in the details report.

Suite MemoryManager read,write

MemoryManager reading and writing.

read (100 bytes)Reads a block of 100 bytes.
read (1000 bytes)Reads a block of 1000 bytes.
write (100 bytes)Writes a block of 100 bytes.
write (1000 bytes)Writes a block of 1000 bytes.

Input ranges from 0 to 1000 in 100 steps.
